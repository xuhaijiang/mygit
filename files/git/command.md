# Git
### git 初始化
`git init` 

### git 状态
`git status`

### git 添加文件
`git add readme.md`

### git 提交文件
`git commit -m "本次提交说明"`

### git stage控件比较
`git diff --staged`

### git 回退
`git reset readne.md`

### git 提交到远程 [master](https://github.com/xuhaijiang/mygit.git) 分支
`git push origin master`

### git 删除
`git rm "*.txt"`
